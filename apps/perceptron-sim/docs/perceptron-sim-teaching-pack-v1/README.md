# Perceptron Sim — Teaching Pack (v1)

This pack contains:
- **docs/labs/**: six 5–10 minute guided labs.
- **scripts/**: two short video scripts (Ep1–Ep2).
- **book/**: outline + two starter chapters and a simple Pandoc build.

## Suggested order
1. Record Ep1–Ep2 using the Sim and the scripts here.
2. Run Labs 1–3; capture screenshots for the book.
3. Draft Chapters 3–6 using the lab outputs.

## Build book (optional)
Requires `pandoc` installed:

```bash
cd book
make html
# or
make pdf
```
